Tml API Plugin For Wordpress
=====================

This plugin adds an extra layer of the API to integrate with WPML and Polylang localization plugins.


Documentation
==================

Please visit https://translationexchange.com/docs/plugins/wordpress for the documentation on how 
to install and configure the plugin.
