Translation Exchange API for your WordPress site.

== Description ==
This API plugin allows Translation Exchange to integrate with your localization plugin in minutes.

Import content, translate and publish it right back.


== Installation ==

1.  Install and activate this plugin.
1.  [Sign up for a free Translation Exchange account](https://dashboard.translationexchange.com/signup)
1.  Link your account with WordPress.
1.  Create a new project.
1.  Import pages, posts, translate them and publish them right back

= Minimum Requirements =
* WordPress 3.0 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater
